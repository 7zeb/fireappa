# Bare Server Setup Guide
If you want to host your own version of **Fracital** follow these steps
```
$ git clone https://github.com/titaniumnetwork-dev/Ultraviolet-Node --recursive
$ cd Ultraviolet-Node
$ npm install
$ npm start 
```
